# 📚 Educator - Online Learning Platform

A modern, fully responsive online education platform built with Next.js 15 and Tailwind CSS v4.

🔗 **[Live Demo](https://educator-wzqv.vercel.app/)**

---

## Tech Stack

- **Next.js 15** - App Router
- **TypeScript**
- **Tailwind CSS v4**
- **shadcn/ui**
- **Lucide React**
- **Vercel** - Deployment

---

## Getting Started

```bash
git clone https://github.com/MrBrain314/educator.git
cd educator
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
